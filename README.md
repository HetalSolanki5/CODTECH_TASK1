Overview of the project
A simple calculator is a basic application that allows users to perform fundamental arithmetic operations like addition, subtraction, multiplication, and division. The project will help you understand how to work with Python functions, input/output, conditionals, and loops. Below is an overview of what such a project might include:

1. Objective
Create a simple text-based calculator that performs the following arithmetic operations:
Addition
Subtraction
Multiplication
Division
Optionally, modulus, exponentiation, and floor division.
2. Key Features
User Input: Users should be able to enter two numbers and select an operation.
Basic Arithmetic Operations: Support for addition, subtraction, multiplication, and division.
Error Handling: Handle errors like division by zero or invalid inputs.
Looping: The program should continue to run until the user chooses to exit.
Exit Option: Allow users to quit the calculator.
